# Grita Karaoke Clone

A static clone of the Grita Karaoke website (gritakaraoke.com).

## Features

- **4 Pages**: Home, Menu, About, Contact/Reserve
- **Responsive Design**: Mobile, tablet, and desktop breakpoints
- **Animations**: Scroll-triggered fade-ins, infinite image carousels
- **Dark Theme**: Deep navy background with gold accents

## Tech Stack

- Pure HTML5, CSS3, JavaScript (no frameworks)
- CSS Custom Properties for theming
- Intersection Observer API for scroll animations
- Google Fonts (Forum, Inter Display)

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Homepage with hero video, image carousel, reserve section |
| `menu.html` | Menu page with drinks, food, desserts |
| `about.html` | About/story page |
| `contact.html` | Reservation form and contact info |

## Local Development

Simply open `index.html` in a browser, or serve with any static server:

```bash
# Python
python -m http.server 8000

# Node.js
npx serve
```
